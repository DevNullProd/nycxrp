Blockchain

Blockchain means many different this but at it's core it's a peer-to-peer (P2P) network, not that unlike BitTorrent

Even the prodigal Blockchain Bitcoin shares its namesake with BitTorrent

In the case of Blockchain the "file" being shared is an application specific database

BTC

With Bitcoin, new records are added to that database by mining, a resourse-intensive software operations

Upon conclusion a miner is granted a certain amount of BTC and

faciliates the execution of transactions modifying balances

With XRP the shared database consists of

  • The state tree - a record of all accounts, balances, settings, etc
  • Transaction tree - the full history of changes to all accounts

XRP doesn't entail mining, all XRP was included in the DB (the "ledger") on genesis

The protocol & software implement a "consensus" process where transactions are ordered

Prevents double spending!

Thus we get all the same benefits of Proof Of Work (PoW) but without the energy use!

Other benefits:

  • Deterministic Consensus Time (PoW is probabilistic)

  • Developer & financial friendly ecosystem

  • Self-governance via a protocol amendment system
